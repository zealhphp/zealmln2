Sambutan Kepala Sekolah

Sugeng tanggap warsa, Wr. Wb.

Puji syukur konjuk dhumateng ngarsanipun Allah SWT, ingkang sampun paring rahmat saha hidayahipun, satemah saged damel website SMK Bhakti Praja Batang menika. Kanggo maju pendidikan ing jaman perkembangan teknologi informasi sing cepet iki, macem-macem prasarana sing kondusif penting, lan kita ngupayakake kanggo nggayuh iki kanthi maksimal. Situs web iki nduweni tujuwan kanggo menehi platform kreatifitas lan informasi kanggo siswa SMK Bhakti Praja Batang. Kanthi luwih jembar, situs web iki nduweni tujuwan kanggo ngenalake, menehi informasi, lan menehi wawasan marang masyarakat umum babagan SMK Bhakti Praja Batang.
Kangge panyebaran situs web punika, ngaturaken agunging panuwun dhumateng: Dinas Pendidikan dan Kebudayaan Provinsi Jawa Tengah, Dinas Pendidikan Provinsi Jawa Tengah Cabang XIII, Rapat Kerja Kepala Sekolah (MKKS) SMK Kabupaten Batang, para guru saha staf administrasi SMK Bhakti Praja Batang, saha para siswa SMK Bhakti Praja Batang.

Muga-muga situs web iki bisa migunani kanggo kabeh pihak sing terlibat ing pendhidhikan lan pemantau pendidikan, utamane sing ana gandhengane karo SMK Bhakti Praja Batang. Kita tansah nampi saran lan masukan saka macem-macem pihak kanggo nganyari awake dhewe, supaya isi lan kualitas situs web kita terus berkembang lan nambah. SMK Bhakti Praja Batang, "Asor lan Prihatin."

Tentrem rahayu.

Nuwun sewu,

Kepala SMK Bhakti Praja Batang

Visi

Terwujudnya SMK yang menghasilkan Sumber Daya Manusia ber-Akhlaq mulia, Kompeten, Berdaya saing global, Andal dan memiliki Ragam keunggulan (AKBAR).

Misi

  1. Nerapake nilai-nilai iman lan taqwa ing kabeh kegiatan sekolah.
  2. Nyengkuyung laku ngibadah miturut agamane saben wong.
  3. Ningkatake fasilitas lan lingkungan sinau sing aman lan nyaman sing nyukupi standar pendidikan nasional lan standar kerja industri saka segi kualitas lan kuantitas.
  4. Nglatih siswa ing saben bidang kompetensi selaras karo perkembangan ing sektor industri.
  5. Mbangun kemitraan karo institusi sing relevan ing njero lan luar negeri.
